## Chatbot with API git
This project is a chatbot, developed using the Take Blip builder. And also a REST API (.NET) that integrates with the git API and provides data to the chatbot.


## Technology 

* Javascript
* ASP.NET core 6.0

## Services Used

* Blip Builder
* Heroku
